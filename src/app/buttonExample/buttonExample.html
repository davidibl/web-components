Standard Buttons

Den Button gibt es in unterschiedlichen Varianten. Als Secondary oder Primary Button, als Fliegender Button und als Button der einen Spinner anzeigen kann.

Über das API kann ein Button deaktiviert werden.

<xn-button [default]="true">Default Button</xn-button>
