Tabs

Die Tabs Komponente ermöglicht die Darstellung von umschaltbaren Tabs. Die zugehörige Menüleiste wird hierbei durch die Komponente erstellt. Jedes Tab Element ist eintweder ein xn-tab oder leitet von TabComponent ab.

Das Styling kann über eine Anpassung der SCSS Variablen vorgenommen werden.

Titel und Icon für das Mneüband können über API mitgegeben werden. Bei abgeleiteten Elementen können die korrespondierenden Properties auch direkt gesetzt sein. 

<xn-tabs>
    <xn-tab [title]="'Tab A'">
        Content
    </xn-tab>
    <xn-tab [title]="'Tab B'">
        <h2>Other Content</h2>
    </xn-tab>
</xn-tabs>
