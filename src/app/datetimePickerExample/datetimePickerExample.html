Datumseingabe

Der Datetimepicker kann Eingaben als Text entgegen nehmen oder mit einer Kalenderauswahl verwendet werden. Um die Kalenderauswahl zu öffnen muss das Kalender Icon angeklickt werden.

Das API und die Möglichkeiten zu stylen teilt sich die Komponente mit allen anderen Eingabefeldern. Ein Date Objekt wird als gebundenes Datenmodell erwartet und auch wieder zurück gegeben. 

<xn-datetime-picker [(ngModel)]="currentDate"
    [placeholder]="'placeholder'"
    [label]="'label'">
