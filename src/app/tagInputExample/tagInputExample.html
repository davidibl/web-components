Tag Eingabe

Das Tag-Eingabe Element ermöglicht die Eingabe mehrerer abgegrenzter Texte.

Die Komponente unterstützt Standardvalidatoren. So kann Beispielsweise validiert werden das mindestens eine Angabe gemacht wurde.
Über ngModel wird ein String-Array rein und raus gebunden.

Das Styling der Komponente übernimmt bei der Eingabebox das Styling der Standard-Texteingabe. Alle anderen Parameter können über SCSS Variablen gesteuert werden. 

<xn-tag-input [(ngModel)]="tags"></xn-tag-input>
