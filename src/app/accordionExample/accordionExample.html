Accordion

Das Accordion wird verwendet um einacfher Workflows im UI abzubilden. Hierbei kann zur Entwicklungszeit festgelegt werden ob alle Schritte der Reihe nach oder durcheinander erledigt werden können.

Das Styling kann über eine Anpassung der SCSS Variablen vorgenommen werden.

Das Accorion setzt sich aus einer äußeren Komponente xn-accordion und den Accordion Schritten xn-accordion-element zusammen.

Die einzelnen Schritte nutzen den Validitätsstatus jedes enthaltenen Formulars und/oder eines über Parameter übergebenen zusätzlichen Validitätsparameters. 

<xn-accordion [preventStepWhenInvalid]="false" [linearStepping]="true">
    <xn-accordion-element [title]="'Schritt 1'">
        <div step-body>
            <p>Test 1</p>
        </div>
    </xn-accordion-element>
</xn-accordion>
