Workflow

Die Workflow Komponente verbindet Workflow Navigation und Accordion zu einem UI Worflow Element.

Das Styling kann über eine Anpassung der SCSS Variablen vorgenommen werden.

Das API der Komponente gibt den aktuellen Workflow State aus. 

<xn-workflow (workflowStateChange)="onWorkflowState($event)">
    <xn-accordion [preventStepWhenInvalid]="true">
        <xn-accordion-element [title]="'Schritt 1'">
            <div step-body>
                <p>Test 1</p>
            </div>
        </xn-accordion-element>
        <xn-accordion-element [title]="'Schritt 2'">
            <div step-body>
                <form #step2form="ngForm">
                    <xn-input-form [label]="'testinput'" name="test" [(ngModel)]="bla" xnRequired></xn-input-form>
                </form>
            </div>
        </xn-accordion-element>
    </xn-accordion>
</xn-workflow>
