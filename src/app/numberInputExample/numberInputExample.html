Zahleneingabe

Eingabefelder für Zahlen können innerhalb von Angular Formularen oder Standalone verwendet werden.

Das Styling kann über eine Anpassung der SCSS Variablen vorgenommen werden. Die Eingabefelder erwarten als gebundenen Wert eine Zahl und geben auch immer nur Zahlen zurück. Gebunden werden Werte wie gewohnt über Model.

Alle Eigenschaften bezüglich Styling Validierungsfehleranzeige und API sind zum Texteingabefeld identisch. 

<xn-number-input-form [(ngModel)]="number"
    [label]="'Eine Zahl'"
    [name]="'krasseZahl'"></xn-number-input-form>

Einfache Zahleneingabe

Die eingegebene Zahl ist: {{number}}
