Dialoge

Die Dialog Komponente wird verwendet um Modale Dialoge anzuzeigen. Hierbei wird der Hintergrund automatisch verdunkelt. Ob ein Klick ausserhalb des Dialogs den Dialog schließt kann explizit festgelegt werden.

Das Styling kann über eine Anpassung der SCSS Variablen vorgenommen werden.

Die Dialogkomponente stellt Standardmäßig einen Abbrechen- einen OK-Knopf und ein X zum schließen dar. An alle drei Events kann man sich dediziert binden. Es ist möglich alle Buttons auszublenden und eine vollständige Knopfleiste mitzugeben. Der Dialog nimmt einen State entgegen anhand dessen sich steuern lässt ob der OK Button aktuell gedrückt werden darf.

Inhalte wie Titel und Content werden über HTML Elemente projeziert. 

<xn-dialog [(open)]="simpleDialogOpen" [showDefaultButtons]="true">
    <span class="dialog-title">Dialogtitel</span>
    <span class="dialog-body">Content</span>
</xn-dialog>
