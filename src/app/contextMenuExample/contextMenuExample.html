Kontextmenü

das Kontextmenü kann über eine Direktive direkt an jedes Element gebunden werden. Das jeweilige Element hat dann ein eigenes Rechts-Klick Menü. Der Direktive wird hierbei ein Datenkontext übergeben, der bei jedem Klick innerhalb des Kontextmenüs mit ausgegeben wird.
So kann ein Kontextmenü etwa in einer Tabelle oder Liste verwendet werden und liefert das Element auf das geklickt wurde direkt mit.

Das Styling kann über eine Anpassung der SCSS Variablen vorgenommen werden.

Das API erwartet für jeden Menüeintrag ein Command das auf Klick zusammen mit dem Datenkontext wieder ausgesendet wird.

Die Direktive an einem Element

<h3 [xnContextMenuConnect]="contextmenu" [context]="1">Irgendwas</h3>

Die Definition des Menüs

<xn-context-menu #contextmenu (buttonClick)="onContextMenuItemClick($event)">
    <xn-context-menu-item [command]="'alert'">
    <xn-icon class="mr-1" [iconClass]="'fa-shopping-cart'"></xn-icon>Alert Datenkontext</xn-context-menu-item>
</xn-context-menu>
