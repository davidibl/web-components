Listenselektoren

Die Listenselektoren können verwendet werden um die Auswahl aus einer Reihe von Möglichkeiten zu ermitteln. Das Element unterstützt die Bindung eines Custom Templates zur Darstellung der Elemente. Zudem kann definiert werden ob nur ein einzelnes oder mehrere Elemente gewählt werden dürfen und Standard Validatoren werden unterstützt.

Über ngModel wird das Model gebunden. Die Items werden seperat gebunden. Ein Array an selektierten Items wird nach jeder Selektionsänderung wieder ausgegeben.
Die Anzige und die Definition der Dateneigenschaft die gebunden werden sollen können analog der Combobox konfiguriert werden so das die direkte Bindung komplexer Objekte unterstützt wird. 

<xn-list-selector [items]="items"
    xnNotEmpty
    name="selctionA"
    [(ngModel)]="selectedItems"
    [displayPropertyPath]="'name'">
</xn-list-selector>
